## What Must Not Be Done

The active Hessian must not be made positive by adding arbitrary quadratic

## Allowed Resolution Routes

### Route A: Response Operator, Not Energy Hessian

Interpret the active Hessian as a response operator generated by eliminating a
branch variable, not as the positive energy Hessian of the microscopic cell.
Then the stability functional must be a separate positive object whose reduced
linear response reproduces the active witness.

Required next result:

```text
positive microscopic energy + constrained branch elimination
-> indefinite effective response witness
```

### Route B: Constrained Stability

Treat one or more active coordinates as constrained source/projection
coordinates rather than freely fluctuating physical amplitudes. Stability is
then required only on the reduced physical subspace after constraints are
imposed.

Required next result:

```text
constraint map + reduced physical tangent space
-> positive Hessian on physical variations
-> same projection-essential response witness
```

### Route C: Higher-Order Boundedness

Add higher-order stabilizers that do not change the quadratic witness but make
finite-amplitude excursions bounded. This does not by itself prove local
linear stability, so it is weaker than Route A or Route B.

Required next result:

```text
quartic or compact-domain stabilizer
-> bounded finite-amplitude potential
-> unchanged quadratic response witness
```

### Route D: Dynamical Stability

Specify a kinetic or symplectic parent dynamics in which an indefinite response
operator can still produce bounded modes. This route must supply actual mode
frequencies or a conserved positive norm.

Required next result:

```text
kinetic/symplectic structure + response Hessian
-> bounded linear modes
-> no target-score inputs
```
