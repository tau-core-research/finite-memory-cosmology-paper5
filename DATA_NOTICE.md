# Data Notice

This repository contains derived, publication-facing tables and figures for the
Paper 5 finite-memory cosmology method-note package.

Raw Pantheon+, DESI, BOSS/eBOSS, arXiv source packages, downloaded webpages,
private exploratory notes, and unpublished author derivative exports are not
redistributed here. The retained files are derived public-input tables,
source-package text/table extracts, local bootstrap reproduction artifacts, and
audit summaries needed to reproduce the preflight benchmark.

The package should be read as a reproducible diagnostic and measurement-gate
preflight, not as a complete external validation data release and not as an
author-native source export.
